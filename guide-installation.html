Installation

Tout ce qu'il faut pour installer Galac6 sur le Mac de votre enfant. L'installation prend moins de 5 minutes.

Prototype : Galac6 est actuellement disponible uniquement sur macOS (Apple Silicon et Intel). Une version Windows/Linux pourra suivre selon les retours.

Téléchargement

Galac6 est actuellement en bêta privée. Pour recevoir le lien de téléchargement, inscrivez-vous avec votre email :

📩 Demander l'accès à la bêta

Prérequis

  • macOS 10.15 (Catalina) ou supérieur
  • Un compte administrateur sur le Mac
  • Environ 50 Mo d'espace disque

Installer Galac6

  1. Double-cliquer sur le fichier .dmg téléchargé
  2. Une fenêtre s'ouvre montrant l'application Galac6
  3. Glisser Galac6 dans le dossier Applications
  4. Fermer la fenêtre du DMG

C'est tout ! L'installation est terminée. L'application est dans le dossier Applications.

Correction Gatekeeper

Si macOS affiche le message « Galac6 est endommagé et ne peut pas être ouvert », c'est parce que l'application n'est pas signée avec un certificat Apple (version bêta).

Pour corriger cela, ouvrir Terminal (Applications → Utilitaires → Terminal) et exécuter :

xattr -cr /Applications/Galac6.app

Puis relancer Galac6 normalement. Cette commande n'a besoin d'être exécutée qu'une seule fois.

Explication : xattr -cr supprime l'attribut de « quarantaine » que macOS ajoute aux fichiers téléchargés. C'est une procédure standard pour les applications en développement non distribuées via l'App Store.

Premier lancement

Au premier lancement, macOS affiche une fenêtre du Trousseau d'accès :

« galac6 veut utiliser vos informations confidentielles gardées dans "galac6" de votre trousseau »

  1. Saisir le mot de passe de session macOS (celui utilisé pour déverrouiller le Mac)
  2. Cliquer sur « Toujours autoriser »

Trousseau d'accès : Galac6 utilise le trousseau macOS (Keychain) pour stocker les clés API de manière sécurisée. En cliquant sur « Toujours autoriser », vous évitez que cette demande se répète à chaque lancement.

L'application s'ouvre ensuite sur l'assistant de configuration intégré dans Galac6.

Mise à jour

Pour mettre à jour Galac6 :

  1. Télécharger le nouveau fichier .dmg
  2. Ouvrir le DMG et glisser Galac6 dans Applications (remplacer l'ancienne version)

Données conservées : toutes vos données sont conservées lors d'une mise à jour (profils élèves, conversations, configuration, clés API).

Désinstallation

Pour supprimer Galac6 :

  1. Quitter l'application (Cmd+Q)
  2. Glisser Galac6.app depuis /Applications vers la Corbeille

Nettoyage complet (optionnel)

Pour supprimer aussi les données utilisateur :

# Données de l'application
rm -rf ~/Library/Application\ Support/com.galac6.app

# Caches
rm -rf ~/Library/Caches/com.galac6.app
rm -rf ~/Library/WebKit/com.galac6.app

# LaunchAgent
rm -f ~/Library/LaunchAgents/com.galac6.app.keepalive.plist

# Fichiers système (nécessite sudo)
sudo rm -rf /Library/Application\ Support/Galac6
sudo rm -f /Library/LaunchDaemons/com.galac6.appmonitor.plist
sudo rm -f /Library/PrivilegedHelperTools/Galac6AppMonitor

Si le filtre IA était actif, nettoyer le fichier /etc/hosts :

sudo nano /etc/hosts
# Supprimer le bloc entre "# >>> Galac6 AI Blocker" et "# <<< Galac6 AI Blocker"
