Guide d'installation et d'utilisation

Galac6 est un assistant IA conçu pour les collégiens et lycéens (de la 6ème à la Terminale). Tout ce que vous devez savoir pour l'installer, le configurer et l'utiliser.

Installation

Prototype : Galac6 est actuellement disponible uniquement sur macOS. Une version Windows/Linux pourra suivre selon les retours.

Prérequis

  • macOS 10.15 (Catalina) ou supérieur
  • Compte administrateur sur le Mac
  • Clé API d'un fournisseur d'IA (OpenAI, Anthropic ou Google Gemini)

Téléchargement

Télécharger la dernière version (.dmg) ici :
Galac6 — Téléchargement (.dmg)

Procédure

  1. Double-cliquer sur le fichier Galac6_x.x.x_aarch64.dmg téléchargé
  2. Une fenêtre s'ouvre montrant l'application Galac6
  3. Glisser Galac6 dans le dossier Applications
  4. Fermer la fenêtre du DMG
  5. Si macOS affiche « Galac6 est endommagé », ouvrir Terminal et exécuter :
    xattr -cr /Applications/Galac6.app
  6. Ouvrir Galac6 depuis le dossier Applications (ou via Launchpad)
  7. Au premier lancement, macOS affiche une fenêtre du Trousseau d'accès : « galac6 veut utiliser vos informations confidentielles gardées dans "galac6" de votre trousseau ». Saisir le mot de passe de session macOS (celui utilisé pour déverrouiller le Mac) et cliquer sur « Toujours autoriser ». Cela permet à Galac6 de stocker vos clés API de façon sécurisée dans le trousseau macOS.

Trousseau d'accès : Galac6 utilise le trousseau macOS (Keychain) pour stocker les clés API de manière sécurisée. En cliquant sur « Toujours autoriser », vous évitez que cette demande se répète à chaque lancement. Si vous cliquez sur « Autoriser » uniquement, la fenêtre réapparaîtra au prochain démarrage.

Premier lancement — Configuration

Au premier lancement, un assistant de configuration guide l'administrateur (parent).

1. Mot de passe administrateur Galac6

  • Choisir un mot de passe (6 caractères minimum)
  • Ce mot de passe protège l'accès aux paramètres de l'application
  • Il est indépendant du mot de passe macOS

2. Thème

  • Clair, Sombre ou Automatique (suit le réglage du Mac)

3. Clé API

Pour chaque fournisseur :

  1. Saisir la clé API (stockée de manière sécurisée dans le Trousseau macOS)
  2. La clé est testée automatiquement et les modèles disponibles s'affichent
  3. Sélectionner le modèle souhaité
  4. Définir la priorité (1 = principal, 2-3 = secours)

Conseil : Configurer deux fournisseurs permet un basculement automatique si le quota du premier est épuisé.

Gestion des élèves

Ajouter un élève

Depuis l'écran d'accueil, cliquer sur + pour ajouter un élève :

  1. Prénom de l'élève
  2. Langue préférée (Français, Anglais, Espagnol, Allemand, Italien, Portugais)
  3. Âge (5 à 23 ans)
  4. Niveau scolaire (suggéré automatiquement d'après l'âge)

Niveaux disponibles (système français) :

  • Primaire : CP, CE1, CE2, CM1, CM2
  • Collège : 6ème, 5ème, 4ème, 3ème
  • Lycée : Seconde, Première, Terminale
  • Supérieur : Bac+1 à Bac+5

Supprimer un élève

Dans le panneau administrateur > Élèves, cliquer sur l'icône de suppression à côté de l'élève.

Utilisation par l'élève

Écran d'accueil

L'élève sélectionne son profil sur l'écran d'accueil, puis accède à son tableau de bord.

Chat avec l'IA

L'élève peut poser des questions à l'assistant IA adapté à son niveau :

  • Matières disponibles : Mathématiques, Français, Histoire-Géographie, Physique-Chimie, SVT, Anglais, Espagnol, Allemand, Philosophie, SES, Informatique
  • Onglets : jusqu'à 8 conversations simultanées
  • Historique : les conversations sont sauvegardées et accessibles depuis le tableau de bord

Actions rapides

Des boutons permettent de démarrer rapidement :

  • Explique-moi…
  • Aide-moi à résoudre…
  • Résume…
  • Lancer un quiz
  • Lancer une dictee
  • Générer une fiche de révision

Éditeur de formules mathématiques

Pour les matières scientifiques, un éditeur de formules est intégré :

  1. Cliquer sur le bouton fx à gauche de la zone de saisie
  2. Un champ WYSIWYG apparaît avec un clavier mathématique virtuel
  3. Composer la formule (fractions, exposants, racines, lettres grecques…)
  4. Cliquer sur ✓ pour insérer la formule dans le message

Coller une image

L'élève peut coller une image (Cmd+V) dans le chat pour que l'IA l'analyse (photo d'un exercice, graphique…).

Mode Quiz

Le mode quiz permet à l'élève de s'auto-évaluer :

  1. Choisir une matière et un sujet
  2. Choisir le nombre de questions : 3, 5 ou 10
  3. L'IA génère un quiz adapté au niveau de l'élève

Types de questions

  • QCM (choix multiples) — correction instantanée
  • Questions ouvertes — évaluées par l'IA avec un retour détaillé

Résultats

  • Score affiché (ex : 4/5 — 80%)
  • Correction détaillée pour chaque question
  • Explication des erreurs
  • XP gagnés :
    • 20 XP par bonne réponse QCM
    • 30 XP par bonne réponse ouverte
    • +50 XP bonus si le score dépasse 80%

Mode Dictee

Le mode dictee propose un exercice de dictee complet, de la lecture a la correction :

Configuration

L'eleve choisit entre deux sources :

  • Dictee generee par l'IA : choisir un theme et un sujet, l'IA genere un texte adapte au niveau de l'eleve
  • Dictee personnalisee : le parent colle un texte, ou colle/depose une image (screenshot) dont le texte est extrait automatiquement par l'IA

Deroulement

Le texte est decoupe en groupes de souffle naturels (5 a 10 mots). Chaque segment est :

  1. Lu une premiere fois a vitesse normale
  2. Pause de 5 secondes pour ecrire
  3. Relu une seconde fois plus lentement
  4. Pause de 8 secondes, puis passage automatique au segment suivant

La ponctuation est dictee a voix haute (« virgule », « point », « point d'interrogation », etc.).

L'eleve peut appuyer sur Pause a tout moment pour arreter temporairement la dictee, puis Reprendre quand il est pret.

Relecture

Apres la derniere phrase, l'eleve accede a un ecran de relecture ou il peut :

  • Relire et corriger l'ensemble de son texte
  • Reecouter la dictee en entier
  • Soumettre quand il est satisfait

Correction et notation

L'IA corrige la dictee avec un bareme adapte au niveau :

  • Note sur 10 avec penalites par type de faute :
    • Grammaticale (accords, conjugaison) : -0,5 a -1,5 pt selon le niveau
    • Lexicale (orthographe d'usage) : -0,25 a -0,5 pt
    • Mineure (accent, trait d'union) : -0,25 pt
  • Copie corrigee : le texte complet s'affiche avec les erreurs en rouge barre et les corrections en vert
  • Detail des erreurs : chaque faute est expliquee avec la regle de grammaire ou d'orthographe correspondante
  • Commentaire global encourageant de l'IA

Historique et progression

Sur l'ecran de configuration, l'eleve peut consulter :

  • Un graphique de progression des notes sur les dernieres dictees
  • Un tableau d'historique avec la date, la note et le detail des fautes (grammaticales, lexicales, mineures) pour chaque dictee

XP gagnes

  • 20 XP de base par dictee
  • +50 XP si note ≥ 8/10
  • +30 XP si note ≥ 6/10
  • +15 XP si note ≥ 4/10

Tableau de bord

Le tableau de bord de l'élève affiche :

Niveau et XP

  • Niveau actuel (100 XP par niveau)
  • Barre de progression vers le niveau suivant
  • XP total accumulé

Statistiques

  • Messages envoyés (total)
  • Quiz complétés
  • Meilleur score de quiz
  • Dictees complétées
  • Série actuelle (jours consécutifs d'utilisation)
  • Meilleure série

Matières travaillées

Répartition visuelle des matières étudiées avec pourcentages.

Conversations récentes

Filtre URL (blocage de sites IA)

Bloque l'accès aux sites d'IA concurrents via le fichier /etc/hosts :

  • Activé / Désactivé d'un clic
  • 300+ domaines bloqués par défaut (ChatGPT, Claude, Gemini, Perplexity, solveurs mathématiques, sites d'aide aux devoirs…)
  • Domaines personnalisés : ajouter ou retirer des domaines
  • Découverte automatique : détecte mensuellement de nouveaux services d'IA
  • Redémarre automatiquement les navigateurs pour appliquer les changements

Note : Le filtre est désactivé par défaut. Le parent l'active quand il le souhaite.

Mise à jour

Pour mettre à jour Galac6 :

  1. Télécharger le nouveau fichier .dmg
  2. Ouvrir le DMG et glisser Galac6 dans Applications (remplacer l'ancienne version)
  3. Les données sont conservées (élèves, conversations, configuration, clés API)

Désinstallation

Pour supprimer Galac6 :

  1. Quitter l'application (Cmd+Q)
  2. Glisser Galac6.app depuis /Applications vers la Corbeille
  3. (Optionnel) Supprimer les données utilisateur :
# Données de l'application (par utilisateur)
rm -rf ~/Library/Application\ Support/com.galac6.app

# Caches
rm -rf ~/Library/Caches/com.galac6.app
rm -rf ~/Library/WebKit/com.galac6.app

# LaunchAgent keepalive
rm -f ~/Library/LaunchAgents/com.galac6.app.keepalive.plist

# Fichiers système (nécessite sudo)
sudo rm -rf /Library/Application\ Support/Galac6
sudo rm -f /Library/LaunchDaemons/com.galac6.appmonitor.plist
sudo rm -f /Library/PrivilegedHelperTools/Galac6AppMonitor

4. (Optionnel) Nettoyer le filtre URL si actif :

sudo nano /etc/hosts
# Supprimer le bloc entre "# >>> Galac6 AI Blocker" et "# <<< Galac6 AI Blocker"
